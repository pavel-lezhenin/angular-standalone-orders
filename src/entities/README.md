# Entities

Domain models, data models, and data access services.

## Structure

Each entity domain folder contains:
- `*.model.ts` - Data models and interfaces
- `*.service.ts` - Data access service (API calls)
- Type definitions specific to this domain

## Examples

- `user/` - User model and user API service
- `order/` - Order model and order API service
- `product/` - Product model (if needed)
